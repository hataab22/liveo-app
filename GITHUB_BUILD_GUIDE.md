# 🚀 بناء Liveo APK عبر GitHub Actions

---

## 📋 الخطوات الكاملة (10 دقائق)

---

### 1️⃣ إنشاء Repository على GitHub

1. **اذهب لـ:** [github.com](https://github.com)

2. **سجل دخول** (أو أنشئ حساب إذا ما عندك)

3. **اضغط على +** (أعلى اليمين)

4. **اختر: New repository**

5. **املأ المعلومات:**
   ```
   Repository name: liveo-app
   Description: Liveo IPTV Application
   Public ✅ (أو Private إذا تبي خاص)
   ```

6. **اضغط: Create repository**

---

### 2️⃣ رفع المشروع على GitHub

#### الطريقة الأسهل (عبر الواجهة):

1. **في صفحة Repository الجديد، اضغط:**
   ```
   uploading an existing file
   ```

2. **اسحب مجلد LiveoApp كامل** (أو اضغط choose your files)

3. **انتظر حتى ترفع كل الملفات** (قد ياخذ دقيقتين)

4. **اضغط: Commit changes**

---

### 3️⃣ تفعيل GitHub Actions

**GitHub Actions سيشتغل تلقائياً!**

بعد رفع الملفات:

1. **اذهب لتبويب Actions** (أعلى الصفحة)

2. **ستشوف: Build Liveo APK** يشتغل الآن 🟡

3. **انتظر 3-5 دقائق** حتى يكتمل

4. **لما يخلص سيصير:** ✅ **Success!**

---

### 4️⃣ تحميل APK

**بعد ما يخلص البناء:**

1. **اضغط على Build** (اللي نجح)

2. **نزّل للأسفل** في الصفحة

3. **تحت "Artifacts" ستجد:**
   ```
   📦 liveo-apk
   ```

4. **اضغط عليه → سيحمّل APK!** 🎉

---

### 5️⃣ تثبيت APK على التلفزيون

1. **فك ضغط الملف** (سيطلع app-debug.apk)

2. **انقله للتلفزيون:**
   - USB
   - Google Drive
   - أي طريقة

3. **في التلفزيون:**
   - Settings → Security → فعّل Unknown Sources
   - File Manager → اختر APK
   - Install

✅ **مثبّت!**

---

## 🔄 إعادة البناء (إذا عدّلت الكود)

**في أي وقت تبي تبني APK جديد:**

1. **اذهل لـ Actions** في GitHub

2. **اضغط: Build Liveo APK** (من الجانب الأيسر)

3. **اضغط: Run workflow** (زر رمادي على اليمين)

4. **اختر Branch: main**

5. **اضغط: Run workflow** (الزر الأخضر)

**سيبني APK جديد! ✅**
